### Project Name:

**Smart Appointment Scheduling System**

### One-line Description:

A full-stack MERN application that centralizes appointment booking across healthcare, business, and technology services with secure multi-level authentication and role-based access control.

### Tech Stack:

**React.js, Node.js, Express.js, MongoDB Atlas, HTML5, CSS3, JavaScript, REST APIs, Vercel, Render, Git, GitHub**

### GitHub Link:

**https://github.com/Jagan-2406/Appointment_Scheduling**

### Live Link:

**https://appointment-scheduling-zeta.vercel.app/**

### Key Outcome:

* Built a production-ready full-stack appointment scheduling platform using the MERN stack.
* Implemented secure multi-level authentication (Admin, Service Provider, Customer).
* Developed centralized appointment management for multiple service categories.
* Successfully deployed the application using **Vercel**, **Render**, and **MongoDB Atlas**.
